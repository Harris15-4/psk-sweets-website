# PSK Sweets Website 🍬

A modern and responsive business website developed for a sweets brand to enhance their online presence and simplify customer ordering.

## 🚀 Features

* Clean and premium UI/UX design
* Product categorization and listing
* Custom order system (date, time, branch selection)
* WhatsApp-based ordering integration
* Fully responsive (mobile + desktop)

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* React (if used)

## 🎯 Purpose

To help local businesses go digital and improve customer engagement through modern web solutions.
